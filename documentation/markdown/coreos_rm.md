## coreos rm

deletes CoreOS image locally

### Synopsis


deletes CoreOS image locally

```
coreos rm
```

### Options

```
      --channel="alpha": CoreOS channel
  -h, --help[=false]: help for rm
      --old[=false]: removes outdated images
      --version="latest": CoreOS version
```
